# What-To-Do
一款解决无聊时“接下来要去做些什么”的微信小程序，仅供学习参考。

# 实现的功能
```

 随机抽选（防止选择强迫症）

 使用百度地图api获取所在定位查询天气

 使用腾讯视频插件播放视频
 
 获取酷狗音乐华语新歌并实现播放

 利用加速度传感器实现检测手机是否反扣，让用户放下手机，专心学习

 使用百度地图api获取所在定位附近的商场

 部分功能有待增加和完善...

 UI有待完善...
 
```
